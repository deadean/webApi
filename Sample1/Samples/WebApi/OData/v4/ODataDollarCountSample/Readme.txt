ODataDollarCountSample
-----------------------------------------------------------------------------

This sample shows how to use the path segment $count in Web API for OData v4.0. The sample covers:

1) $count follows a collection of primitive type.
2) $count follows a collection of Complex type and is followed by the query options $filter.
3) $count follows a collection of Enum type property which is decorated by [NotCountable].

This sample is provided as part of the ASP.NET Web Stack sample repository at 
http://aspnet.codeplex.com/
